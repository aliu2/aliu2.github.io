About Me

I'm a third-year Computer Applications and Software Engineering student at Dublin City University. Over the course of my studies here I've developed numerous skills that have helped me to achieve and maintain a 2.1 GPA. I love solving problems, getting to know new people and jumping outside of my comfort zone to try new things!

Python

This was the very first programming language I learned. It's easy-to-understand syntax and readability made it very easy to focus on solving problems while programming, rather than just focusing on the code itself. I love how the language is so simple yet so versatile and it's currently one of my favorite languages to use.

Java

If I was to pick an all-time favorite programming language, it would be Java. It's object-oriented approach and strict typing makes it a very intuitive language for me as I love to solve problems in an object-oriented manner. It also taught me a lot about the general semantics that many programming languages follow.

JavaScript

I think the most fun language to learn for me was probably JavaScript. I used it to animate web pages and create games like Snake. Seeing things jump and move around on the screen and respond to your keystrokes is a very different experience to coding things like a Sentiment Analysis bot in Java but different in a good way. The syntax is very similar to Python so it was very easy to pick up and start creating things straight away.

HTML, CSS & Bootstrap

HTML and CSS were a little more tricky to learn as they aren't conventionally interpreted or compiled, so debugging was more difficult. However, as I got more used to the structure and using the Chrome Inspector, it became less of an issue and I enjoyed using the languages more and more. When I learned the Bootstrap framework, everything was made so much easier and more consistent, with fewer bugs and compatibility issues and it eventually led to me creating this very website!
